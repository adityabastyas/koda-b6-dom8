# Progam Konversi Suhu

App ini menjalankan progam konversi suhu dari ceclcius ke fahrenheit, celcius ke reamur dan celcius ke kelvin

![](image.png)

## Cara Menjalankan App

- buka index.html
- masukkan angka ke dalam form input
- klik submit
- maka akan memunculkan tabel baru berupa no urutan dan angka yang sudah di konversikan dan angka celcius awal
- ketika menginputkan angka baru dan klik submit, makan tabel akan bertambah ke bawah sesuai urutan
- selesai
